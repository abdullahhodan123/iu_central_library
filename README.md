# 📚 Central Library IU

A Django-based web application designed to manage library services efficiently.
This system allows students to search books by department, request loans, and interact with books through comments, while admins handle approvals and management.

---

## 🚀 Features

* 🔐 User registration & authentication system
* 🔍 Search books by department
* 📖 Book details with comments
* 🛒 Book loan request system
* ✅ Admin approval for loan requests
* 📄 PDF receipt generation after approval
* 👤 User profile management

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Tools:** Git, GitHub

## ⚙️ Installation

Follow the steps below to run this project locally:

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/central-library-iu.git
```

### 2️⃣ Navigate to project directory

```
cd central-library-iu
```

### 3️⃣ Create virtual environment

```
python -m venv env
```

### 4️⃣ Activate virtual environment

**Windows:**

```
env\Scripts\activate
```

**Linux/Mac:**

```
source env/bin/activate
```

### 5️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 6️⃣ Apply migrations

```
python manage.py migrate
```

### 7️⃣ Run the development server

```
python manage.py runserver
```

👉 এখন browser এ যান:
http://127.0.0.1:8000/
